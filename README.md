# Photography NFT Gallery

A curated marketplace for photography NFT collections and fine art photography trading.

## Overview

Photography NFT Gallery provides a premium platform for photographers to showcase and sell their work as NFTs. Features curation services and collector-focused experiences.

## Features

- **Curated Exhibitions**: Professional curation for quality photography
- **Photography Collections**: Support for series and individual pieces
- **Collector Bidding**: Advanced bidding system for fine art
- **Artist Analytics**: Track exhibition performance and sales
- **Gallery Services**: 3.5% curation fee for premium services

## Smart Contract Functions

### Gallery Operations
- `exhibit-photography`: Display photography NFTs for sale
- `purchase-photography`: Buy photography directly
- `place-photography-bid`: Submit bids on photographs
- `accept-collector-bid`: Accept collector offers
- `remove-exhibition`: Remove from gallery

### Gallery Services
- `get-photo-exhibition`: View exhibition details
- `get-collector-bid`: Check bid information
- `get-gallery-stats`: Artist and collector analytics
- `calculate-curation-fees`: Fee breakdown
